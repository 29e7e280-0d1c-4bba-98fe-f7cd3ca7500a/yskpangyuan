# 胖圆

胖圆是一个可以同步头条发布内容的工具，开发中

## 文档

[https://pangyuan.netroby.com](https://pangyuan.netroby.com)

## 使用

【请等待】二进制发布，还在开发中

## 开发

安装git, nodejs, yarn

命令行执行

```bash
git clone https://github.com/netroby/yskpangyuan.git
cd yskpangyuan
yarn install
yarn start
```
